== NAME

os_check - checks the operating system

== SYNOPSIS

*os_check*

== DESCRIPTION

The *os_check(1)* command
checks the operating system.

*1*::
  GNU/Linux

*2*::
  Mac OSX

*3*::
  MSYS

*4*::
  FreeBSD

*999*::
  Unknown

== EXIT STATUS

*0*::
  Sucess.

*1*::
  Failure.
